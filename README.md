## 📝 Sobre o Projeto
O **Numerix // Trilha das Estrelas** é uma plataforma de gamificação educacional focada no ensino de matemática básica. Através de uma interface minimalista, o estudante é desafiado a alcançar a meta aritmética de **10** utilizando o raciocínio estratégico, evitando o erro por ultrapassagem.

> *"A matemática é a linguagem do universo; o Numerix é a trilha para dominá-la."*

## 🎯 Público-Alvo
- **Estudantes do Ensino Fundamental:** Alunos em fase de alfabetização aritmética que precisam reforçar o cálculo mental.
- **Educadores:** Professores que buscam ferramentas digitais interativas para tornar as aulas de matemática mais dinâmicas.
- **Entusiastas da Aprendizagem Lúdica:** Qualquer pessoa que deseje exercitar o raciocínio rápido de forma leve e descontraída.

## 💡 O Que Resolve
O projeto atua diretamente na redução da "ansiedade matemática", oferecendo:
- **Engajamento Ativo:** Transforma a repetição de contas em um desafio estratégico.
- **Feedback Imediato:** Corrige o erro instantaneamente, permitindo que o aluno aprenda com a prática contínua.
- **Redução de Fricção:** Elimina o medo do erro ao proporcionar uma interface segura e recompensadora.

## ✨ Principais Funcionalidades
- **Trilha Dinâmica:** Feedback visual em tempo real conforme as somas propostas.
- **Engine de Recompensa:** Sistema de confetes e feedback de vitória para reforço positivo.
- **UX Pedagógica:** Interface limpa que reduz a carga cognitiva, permitindo foco total no desafio.
- **Acessibilidade:** Desenvolvido com padrões de contraste e usabilidade.

## 🛠 Tecnologias
- **Frontend:** HTML5, CSS3 (Keyframes & Flexbox).
- **Lógica:** Vanilla JavaScript (ES6+).
- **Framework:** Bootstrap 5.3 para estruturação ágil.

## 🚀 Como Executar
O projeto não requer build ou pré-processadores. Siga os passos abaixo no seu terminal:

```bash
# Clone o repositório
git clone https://github.com/FelipeBritoSP10/numerix-path-of-the-stars

# Entre na pasta do projeto
cd numerix-path-of-the-stars

# Abra o arquivo index.html no navegador

# Escolha o comando correspondente ao seu sistema:
open index.html    # macOS
start index.html   # Windows
xdg-open index.html # Linux

```
